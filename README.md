# About Issue tu Gantt

Allow to view your Github Issues on a Gantt Chart

## Installation

- $ git clone
- $ composer install
- $ cp .env-example .env
- $ php artisan key:generate
- edit .env and set GITHUB parameters
- $ php artisan serve
- run your browser on http://localhost:8000/
